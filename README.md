# TidyQuest

TidyQuest is a gamified household chore chart built by [Teda.dev](https://teda.dev). It transforms mundane tasks into a fun RPG-like experience for families, featuring assignments, XP points, rewards, and AI-powered chore suggestions.

## Features

- **Gamified Dashboard**: Track points, leaderboards, and streaks.
- **Chore Management**: Assign recurring tasks with due dates and XP values.
- **Family Profiles**: Custom avatars and colors for each member.
- **Rewards Store**: Redeem XP for real-life rewards (customizable).
- **AI Integration**: Uses local WebLLM to suggest chores.
- **Zero Backend**: All data persists in your browser (LocalStorage).

## Tech Stack

- **HTML5 & Tailwind CSS** (via CDN)
- **jQuery 3.7.x**
- **WebLLM** (Local Browser AI)
- **No Build Tools Needed**

## Usage

1. Open `index.html` to view the landing page.
2. Click "Start Your Quest" to launch the app.
3. Add family members and chores.
4. Complete chores to earn confetti and XP!
